# '몸' 없는 인공지능 
## : 인공지능의 한계와 가능성에 관한 현상학적 고찰
[현상학과 현대철학], 제 104집, 한국현상학회, 2024, 63-93
저자: 최우석(경희대학교 학술연구교수) 

인공지능의 알고리즘적 인지 구조가 인간의 신체적 경험과 상확적 학습을 구현하지 못함 
-> 인간의 행동을 de facto 모방할 순 있어도 그 결과를 도출하는 방식은 근본적으로 다름 

인간은 세계-내-존재로서 주변환경과 상호작용하며 행위를 함 
-> 카오스는 결정적이지만 너무 복잡해 예측할 수 없는 시스템을 의미함
-> 결정적이라면 결국엔 인간의 행동을 인공지능으로 완전히 모방할 수 있지 않을까?
-> 모방은 가능하지만 지능의 동작원리가 근본적으로 다름. 인공지능은 함수처럼 동작하지만 인간은 신체적 감각을 통한 직접적 경험으로부터 결과를 도출함.

인공지능은 임기응변이 어려움

인간의 감각 처리 과정: "내가 차가운 금속을 만질 때, '나'는 금속의 차가움을 느끼는 동시에 '나'의 손가락이 차가움을 감각하는 경험도 한다. 사람은 이와 같은 이중 감각을 통해 자기를 인식하는데, 이처럼 "만지는 행위"로서 감각은 자기를 비롯한 대상을 이해하는 출발점이 된다. "(p.77)
-> 기술적으로 구현할 수 있지 않을까 

인간은 공통된 신체적 특징을 공유해 유비적으로 타인의 인식과 감정을 헤아릴 수 있지만 인공지능은 불가능함 

인공지능은 취약한 신체와 죽음의 공포를 갖지 않는다. 

인간의 인식은 죽어가는 신체의 시간성과 과거 경험의 맥락을 포함한다.

인공지능이 생활세계를 가져도 인간의 것과는 다른 개념 구조를 가질 것.

체화된 인지라고 해도 감각을 처리할 때 시간적으로 완전히 동시적일 수 없음. physical ai에게 감각모듈을 달고 인지모듈과 되먹임을 하도록 설계하면... 주관적 체험도 저장장치로 구현한다면... 다음 세대 로봇에게 유전적으로 지식을 전달할 수 있다면...
