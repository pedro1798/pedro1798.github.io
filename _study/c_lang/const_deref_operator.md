역참조 연산자 \*와 포인터 변수 \*는 완전히 다른 기능을 수행한다.
```c
int i = 10;
const int *p = &i;
*p = 20; // 오류 발생
```
Q. 해당 코드의 3번째 줄에서 오류가 발생하는 이유는?

const int \*p 를 const (int \*p)로 생각한다면, p는 i의 주소 &i를 저장한
상수 변수이기 때문에, i값을 업데이트 해도 p(i의 주소)는 그대로이므로
(형변환이 없는 한) \*p = 20;으로 업데이트가 가능하다고 생각할 수 있다.

하지만 해당 코드는 (const int\*) p로 동작한다. 상수 정수형을 담고 있는 
포인터 p이기 때문에 \*p = 20;으로 업데이트 할 수 없다.

그렇다면 다음 코드는?
```c
int* const p = &i;
p = &j;
```
해당 코드는 int \*를 담고 있는 상수 정수 포인터 자료형 p이기 때문에,
p의 주소값을 &j로 업데이트 할 수 없다.
